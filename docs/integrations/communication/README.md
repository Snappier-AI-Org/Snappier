# Communication Integrations

This section covers communication platform integrations in Snappier.

## Available Platforms

| Platform | Auth Type | Documentation |
|----------|-----------|---------------|
| Slack | OAuth 2.0 | [Guide](./slack.md) |
| Discord | Bot Token | [Guide](./discord.md) |
| Outlook | OAuth 2.0 | [Guide](./outlook.md) |

## Common Use Cases

### Slack
- Send messages to channels
- Send direct messages
- Post rich message blocks

### Discord
- Send messages to channels
- Send embed messages
- Bot interactions

### Outlook
- Send emails
- Read emails
- Calendar events

See individual guides for detailed setup and operations.
