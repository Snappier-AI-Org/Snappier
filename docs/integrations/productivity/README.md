# Productivity Integrations

This section covers productivity tool integrations in Nodebase.

| Tool | Auth Type | Documentation |
|------|-----------|---------------|
| Notion | OAuth 2.0 | [Guide](./notion.md) |
| Trello | OAuth 1.0a | [Guide](./trello.md) |
| GitHub | OAuth 2.0 | [Guide](./github.md) |

## Common Use Cases

### Notion
- Create pages/databases
- Update page properties
- Query databases
- Append content

### Trello
- Create cards
- Move cards between lists
- Update card details
- Add comments

### GitHub
- Create issues
- Update issue status
- Create pull requests
- Add comments

### Todoist
- Create and manage tasks
- Organize projects
- Add comments to tasks
- Create and apply labels
- Filter tasks with natural language

See individual guides for detailed setup and operations.
